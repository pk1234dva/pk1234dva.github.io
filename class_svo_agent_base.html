Detailed Description

Base agent component.

It has two main functionalities:

  1. UpdatePath() method which will query for a path.
  2. Path post processing:
    • The path from SvoManagerMono.FindPath gives us a list of Vector3 positions in the space. However, this path could be choppy - so instead of using the path directly, we do "post processing", searching for the furthest waypoint in this list that can be directly reached without any collisions.
    • This waypoint is stored with the nextWaypointIndex

For custom behaviour, make your own extensions of this class. Here's an example of a simple implementation inside SvoAgent.cs that updates its path regularly, and uses path post processing every frame:

public class SvoAgent : SvoAgentBase
{
[Header("SvoAgent parameters")]
[Tooltip("How often the path is updated")]
[SerializeField] float pathUpdateInterval = 0.2f;
private float pathUpdateTimer = 0.0f;
private void Update()
{
pathUpdateTimer += (Time.deltaTime * Time.timeScale);
if (pathUpdateTimer > pathUpdateInterval)
{
UpdatePath();
pathUpdateTimer = 0.0f;
}
UpdateNextIdx(dueToNewPath: false);
}
private void Start()
{
TrySetStaticManagerReference();
svoManager.CheckValidAgentSize(this.radius);
}
}

Member Data Documentation

◆ managerType

SvoManagerType managerType

SvoManagerType that the agent will try to use.

If using SvoManagerType.Static, agent will attempt to use the global static SvoManagerMono reference. If using SvoManagerType.Instance, you will have to setup a direct reference to a SvoManagerMono component in your scene.

◆ radius

float radius

Radius of the agent used for path finding.

Should be large enough to contain your agent mesh and mesh collider. Used inside path finding, as well as path post processing.

◆ waypointReachedRatio

float waypointReachedRatio

Determines at what distance we consider the agent to have reached the current waypoint.

The distance is determined as radius * waypointReachedRatio. For example, if the radius of the agent is 2.0f and waypointReachedRatio is 1.0f, then once the agent center transform position is 1.0f * 2.0f = 2.0f units away (or less), it will consider the current waypoint as reached, and start aiming towards the next one.

◆ staticEnvironmentLayer

LayerMask staticEnvironmentLayer

LayerMask used for determining collisions in path post processing.

Should most likely be kept at the same value as SvoManagerMono.collisionLayers

◆ postProcessingStepsPerFrame

int postProcessingStepsPerFrame

Amount of path post processing steps that intended to be done regularly, e.g. each frame.

Using path post-processing is generally not necessary if you are frequently making path finding requests, but if not, it's a good idea to keep this above zero and use it in Update or FixedUpdate.
